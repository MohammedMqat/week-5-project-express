# User Stories

1. As an employee, I Want to know info about a lot of anime in one page so I decide fast what to watch and not waste time
   - Can see top 10 anime with image, title, and score
   - Can click a card to see details
2. As a student, I Want to search about anime I like to decide fast what to watch so I spend my free time well
   - Can search for any anime users want
   - Can show clear result with pagination

3. As a youtuber, I want to know full detail of anime I search so I decide which anime I can make video on my channel
   - Can click a card to see full details
